【LeetCode】10. Regular Expression Matching

Posted on 2018-02-08 | In LeetCode

题目描述

Implement regular expression matching with support for '.' and '*'.

题目大意

实现一个正则表达式的检测函数，其中 '.' 可以代表任意一个字符，'*' 代表前面的字符出现了任意次（零次、一次或多次）。

解题思路

在对 isMatch("ab", ".*") → true 的理解上，是 '*' 把 '.' 重复了 $N$ 次；
在对 isMatch("aab", "c*a*b") → true 的理解上，是第一个 '*' 把 'c' 重复了 $0$ 次，第二个 '*' 把 'a' 重复了 $2$ 次；

递归

递归的重点在于对 '*' 的处理上，出现 '*' 表示它前置的字符可出现任意次，所以递归的过程就是不断切割的过程。

动态规划

递归算法会有很多的冗余计算，动态规划实际上就是把这些计算结果存到了状态矩阵里，我们用 $dp[i][j]$ 表示 $s[i:]$ 与 $p[j:]$ 是否匹配，显然，对字符串来说，求解是一个逆序的过程。
