【LeetCode】19. Remove Nth Node From End of List

Posted on 2018-02-18 | In LeetCode

题目描述

Given a linked list, remove the $n^{th}$ node from the end of list and return its head.

Example

Note:
Given n will always be valid.
Try to do this in one pass.

题目大意

删除单项链表中倒数第 $n$ 个节点

解题思路

常规的解法是遍历一次链表得到链表的长度，进而找到倒数第 $n$ 个节点。

如果只遍历一次链表，我们可以使用滑动窗口的思想，构建 $fast$ 和 $slow$ 两个指针，使两个指针之间的距离是 $n$，这样，当 $fast$ 指针指向链表末尾时，$slow$ 指针指向的位置就是倒数第 $n$ 个节点，在实际处理过程中，需要综合考虑链表为空，链表长度为 $1$，以及删除的节点为头节点的情况。
