【LeetCode】15. 3Sum

Posted on 2018-02-12 | In LeetCode

题目描述

Given an array $S$ of $n$ integers, are there elements $a, b, c$ in $S$ such that $a + b + c = 0$? Find all unique triplets in the array which gives the sum of zero.

Note: The solution set must not contain duplicate triplets.

Example

题目大意

给定一个包含 $n$ 个整数的数组，找出其中所有和为 $0$ 的三元组，返回的结果中不能含有重复的三元组。

解题思路

$3Sum$ 问题其实是 $2Sum$ 问题的扩充，固定一个值以后问题就变成了 $2Sum$ 问题，但是如果单纯的使用蛮力的方法，耗时会相当长，所以问题的关键就是如何在避免重复的情况下求解 $2Sum$ 问题。

  • 首先对数据进行排序
  • 初始化两个指针 $j=i+1, k=len-1$，只考虑 $i$ 后面的这部分元素
  • 如果符合要求，需要考虑重复值的情况，根据边界值重新定位 $j$ 和 $k$
  • 如果不符合要求，根据三元组的和的大小来确定改变 $j$ 还是 $k$
